Whack-o-Platypus

Database Handling

Project Type: Individual


Role: Database handling, Game Design

Timeline: 2 Weeks


Tools: Unity, Procreate, Firebase, Visual Studio Code C#

Overview

Task: To create a minigame and implement database features, such as login, sign up, and a leaderboard


In this project, I learned more about handling json data, and linking them to games that I create, which is essential to starting any project. The game is inpired by Perry the Platypus in Phineas & Ferb, and the goal of the game is to whack the platypus as many times as possible, and reach the top of the leaderboard!

Background

In the game, the player can sign into an existing account, or create a new one. Afterwards, they can create unique username for themselves to be used in the game. Once in the game, the player can choose to either start a game, view the leaderboard, or quit. Pressing play will bring the player immediately into the game screen, where the player is required to whack as many platypuses within the time-span of 10 seconds. Afterwards, the player will be brought over to a Game Over screen, where the player can choose to save the score, replay the game, or go to the home page.
