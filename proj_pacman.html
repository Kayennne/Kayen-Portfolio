Pac-Man Game Controller

Controller Design

Project Type: Individual


Role: Handcraft

Timeline: 2 Weeks


Tools: My Hands

Overview

Task: To build a game controller with a MakeyMakey


In this project, I learned about how the MakeyMakey game set works, and how to utilise it to make intriguing game controllers. I was also tasked to identify UI/UX design strengths and weaknesses in the game I chose to make a controller for, and how different game controllers were designed for people.

Background

Who doesn't love Pac-Man? A classic arcade game I'm sure everyone played once when they were younger. One of my favourite things about this game was its simplistic yet addictive game mechanic that gets you hooked to the game. Thus, thinking of a controller idea for this game was really fun to make, from the technicalities, to the design and handycraft, definitely one of my favourite projects!

Design Rationale & Mechanics

Since the game only requires movement in 4 directions (up, down, left, right), the controller I designed has 4 buttons to represent each direction.
