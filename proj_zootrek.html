ZooTrek

Augmented Reality App

Project Type: Individual


Role: App Design, Coding, Gamification

Timeline: 3 Weeks


Tools: Unity, Vuforia, Visual Studio Code C#

Overview

Task: To create an Augmented Reality Mobile Application to provide interaction points for engaging consumers with a brand, product, or person.


This project encouraged me to explore an immersive way of enhancing user experience through Augmented Reality. Making use of various image targets to achieve a desired 3D effect in the app I designed

Background

ZooTrek is a mobile app that enhances users's experience through the witty use of AR technology. Some of its features are:
1. Scanning pictures to know more about a certain animal
2. Interact with 3D models
3. Interactive games
4. Location tracking with AR (map usage)
