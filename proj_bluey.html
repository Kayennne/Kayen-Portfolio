Bluey's First Date

Coding

Project Type: Individual


Role: Coding

Timeline: 4 Weeks


Tools: Unity, Visual Studio Code C#

Overview

Task: To create a basic 3D virtual player experience, by utilizing C# and the Unity Engine. Implement the usage of Unity Physics to collect collectibles, update and display progress through Unity UI, and create controllable player


This project was the first time I was exposed to C# coding, thus it taught me a lot of basics to start with. I was able to code a to-do list for the player to keep track, UI messages that pop up when the player forgets to do something, a collectibles system, and a cute storyline too.

*Models taken from Unity Asset Store

Background

You are a bird named Bluey! Bluey saw a really pretty pink bird flying the other day and wants to ask her out! Help Bluey prepare for his first date by collecting the necessary items in his to-do list and hope for the best!
